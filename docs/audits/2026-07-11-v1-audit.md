# v1 build audit, 2026-07-11

Full audit of the v1 implementation (phases 0-3, built by GPT 5.6 from the phase specs), performed with five parallel review passes (core, db, api, runtime/media, web/player) followed by five parallel fix passes. Every finding below was verified against the code before being fixed. Status: FIXED means repaired and covered by a test where practical; SUPERSEDED means the spec was amended with an explicit supersession note; GAP means enumerated as remaining work, not repaired in this pass.

## 1. Critical (broken product or security)

| # | Area | Finding | Status |
|---|---|---|---|
| C1 | core/config | `z.coerce.boolean()` parsed the strings "false" and "0" as true, so `OIDC_AUTO_PROVISION=false` silently enabled IdP auto-provisioning and `TRUST_PROXY=false` trusted proxy headers | FIXED, strict string parser + tests |
| C2 | core/markers | Resolve marker EDL emitted two timecodes per event with fields on the event line; Resolve cannot import it. This is the flagship differentiator | FIXED, real CMX3600 shape (four TCs, continuation line, ResolveColor name, exclusive out), byte-exact golden tests |
| C3 | worker/media | HDR tonemap filter used `inverse_ootf`, which is not a libplacebo option: every HDR job failed at filter init; and no SDR target colorimetry was set, so even without the bad option the output would be wrongly tagged | FIXED, correct filter chain, mesa-vulkan-drivers added to the worker image |
| C4 | worker/media | Proxy ladder used `scale=WIDTH:-2` with height names (2160/1080/540 treated as widths): every proxy was roughly half resolution | FIXED, `scale=-2:height`, tests assert exact strings |
| C5 | server/worker-pump | 60 second hard timeout on transcode jobs, no heartbeat, and retries started concurrent duplicate ffmpeg runs on the same outputs | FIXED, poll to terminal state with 6h configurable ceiling, heartbeats, worker rejects re-POST of a running job |
| C6 | deploy | Server container ran as root while the worker ran as uid 10001 on the same volume: every job died on EACCES. Compose also shipped fixed public default secrets | FIXED, both containers uid 10001, fail-loud `${SECRET_KEY:?}`, healthchecks, restart policy |
| C7 | db/migrations | Dual D1 migration tracking (wrangler `d1_migrations` vs runtime `__onelight_migrations`): using the documented wrangler flow then deploying re-ran the DDL and 500ed permanently | FIXED, runtime runner reconciles pre-existing schema and is memoized per isolate |
| C8 | player | Loop seeked to the out point (playhead pinned forever); J/K/L: reverse played forward, L after pause played at 0.25x | FIXED, correct transport semantics, reverse emulated by rational-rate stepping |
| C9 | player | `currentTime` determined frame identity (spec forbids it) via an unconditional timeupdate handler with round instead of floor | FIXED, rVFC is the sole source when available, floor-with-epsilon fallback, boundary tests |
| C10 | web | The app never passed the probed rational frame rate to the player (hardcoded 24/1), so frame math was wrong for all non-24fps media | FIXED, version fetch wires num/den and drop_frame through |
| C11 | web/tokens | `packages/web/src/lib/tokens.css` was not the verbatim mockups port (missing the entire neutral scale, palettes, type scale), which caused hardcoded off-neutral colors everywhere | FIXED, byte-identical port, prettierignored to stay verbatim |
| C12 | web/review room | The asset review page used tinted (non R=G=B) chrome, violating the neutral review room hard rule | FIXED, restyled with --n-* tokens only |

## 3. Minor and hygiene

- better-sqlite3 pinned at ^11 has no Node 25 prebuilds; tests failed with NODE_MODULE_VERSION errors that were masked by piped exit codes. Bumped to ^12.11.1. FIXED
- A literal `NUL/` directory of wrangler logs (Windows redirect artifact) sat in the repo root. Deleted, gitignored. FIXED
- `.gitignore` missed packages/web/build/, *.tsbuildinfo, NUL/. FIXED
- CI lacked the docker build job required by phase-0 T23 and the prettier gate. FIXED
- `payload_too_large` was a non-canonical error code; now `validation_failed` with 413 per spec. FIXED
- JSON body cap was bypassable via chunked encoding; part uploads and attachment uploads had no size caps; assembled size was never checked against the declared size. FIXED
- crc32cMatches compared base64 case-insensitively (distinct checksums could match). FIXED
- Local blob store wrote non-atomically (truncated objects on crash). FIXED, temp file + rename
- Server ignored SIGTERM (docker stop hung to SIGKILL). FIXED
- Seed printed the env-supplied admin password to stdout. FIXED
- Worker pump silently no-opped when unconfigured. FIXED, warns once
- wrangler.jsonc: assets lacked SPA not_found_handling; dev SECRET_KEY var now carries a production warning comment. FIXED
- .env.example documented a fraction of the supported variables. FIXED

## Addendum: remainder build, same day

The contract suite, once genuinely exhaustive, surfaced nine additional product bugs, all fixed and regression-tested: sliding session expiry never re-sent the cookie; restricted projects leaked existence via 403; archived projects could not be unarchived; project list pagination could drop visible rows; project managers could not moderate comments; the share reply endpoint accepted any comment id in the database (cross-share and internal-comment leak); re-completing an upload 500ed; GET /admin/jobs leaked jobs across workspaces; /s/* error responses lost the error envelope. Two contract-grade findings were resolved by product work in wave 2: allow_download was stored but enforced nowhere, and watermarked renditions were generated but never served.
