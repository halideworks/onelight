# Landing demo reel

`destination-earth.mp4`: a 12 second excerpt of "Destination Earth"
(1956, John Sutherland Productions for the American Petroleum
Institute), from the Prelinger Archives copy at
https://archive.org/details/4050_Destination_Earth_01_47_33_28.
The film is in the public domain. Re-encoded at its native 30000/1001
rate, 640x480, H.264 CRF 23 with AAC audio; the excerpt starts at
03:56 of the source.
